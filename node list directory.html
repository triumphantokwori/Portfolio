Node List Directory

The Node List Directory is a command line application. It aims to replicate the 'ls' command in Mac or Linux Operating Systems
Here, the index.js file is used to execute the function in the Windows Command Prompt. When run, files and folders in the same directory are logged on the command prompt console.
If a directory is specified relative to the project file path( ' ../ ' is used to move up one directory ), the content of the specified directory is logged. Folders are logged in blue with the use of the npm package - chalk, while files are shown as white labels.
